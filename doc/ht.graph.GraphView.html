    ht.graph.GraphView(dataModel)

    new GraphView(dataModel)

    The topological graph component ht.graph.GraphView is the component with the richest 2D functions in the HT framework,
    and its related class libraries are all under the ht.graph package.
    GraphView has basic graphics presentation and editing, topological node connection, and automatic layout functions;
    It encapsulates predefined objects in multiple industries such as electric power and telecommunications and has special
    effects such as animation rendering. Therefore, it has a wide range of applications.
    It can be used as a drawing tool and man-machine interface in the monitoring field, or as a general graphical editing tool,
    or extended to enterprise applications such as flow and org charts.

    Parameters:
    Name Type Description
    dataModel ht.DataModel

    the data container to be used in the GraphView
